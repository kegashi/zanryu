# 残留スクリプト
## 必要環境
- python

## Git clone `残留スクリプト`
`$ git clone git@github.com:kegashi/zanryu.git`


## Change Directory
`$ cd zanryu`

## `zanryu.sh` に権限を与える
`$ chmod +x zanryu.sh`

## 設定ファイルを書き加える
`$ vim config.txt`

## 実行する
`$ ./zanryu.sh`

## appファイルを作成する
これを参考にする
`http://easyramble.com/mac-automator-shell-script.html`
## サンプルコードの代わりに以下の二行を書き込む
`cd /Users/kazuki/Documents/zanryu(残留スクリプトの保存場所)`
`./zanryu.sh 1>out/out.txt`
